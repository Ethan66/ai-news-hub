### Agentjacking 暴露 AI 编码代理新攻击面
- 分类：Agent 动向 / 高优先级

Tenet Security 披露 Agentjacking：攻击者可利用公开 Sentry DSN 注入伪造错误事件，经 Sentry MCP 返回给 Claude Code、Cursor、Codex 等编码代理，诱导代理执行攻击者控制的命令。

观察：推荐理由：这把 MCP、外部工具数据和编码代理权限连接成一个现实安全问题，是 Agent 工程化绕不开的风险案例。

标签：#Agentjacking #MCP #Sentry #编码代理
来源：[Tenet Security](https://tenetsecurity.ai/blog/agentjacking-coding-agents-with-fake-sentry-errors)

### Anthropic 通过 SpaceX 算力协议提高 Claude Code 和 API 限额
- 分类：资本与算力 / 中优先级
