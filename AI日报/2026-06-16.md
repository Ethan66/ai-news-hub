## 行业动态

### 1. ChatGPT App 月活达到 10 亿

- 分类：企业落地
- 优先级：高优先级
- 摘要：Reuters 援引 Sensor Tower 数据称，ChatGPT App 在 2026 年 5 月达到 10 亿全球月活，成为最快达到这一规模的应用之一；Claude App 同期同比增速约 640%。
- 推荐理由：10 亿月活意味着生成式 AI 已经进入超大规模消费级入口竞争，商业化、监管和竞争都会被重新定价。
- 标签：`ChatGPT` `OpenAI` `Sensor Tower` `消费级 AI`
- 来源：Reuters
- 链接：https://www.reuters.com/technology/chatgpt-app-hits-1-billion-monthly-active-users-record-time-data-shows-2026-06-02

### 2. Agentjacking 暴露 AI 编码代理新攻击面

- 分类：Agent 动向
- 优先级：高优先级
- 摘要：Tenet Security 披露 Agentjacking：攻击者可利用公开 Sentry DSN 注入伪造错误事件，经 Sentry MCP 返回给 Claude Code、Cursor、Codex 等编码代理，诱导代理执行攻击者控制的命令。
- 推荐理由：这把 MCP、外部工具数据和编码代理权限连接成一个现实安全问题，是 Agent 工程化绕不开的风险案例。
- 标签：`Agentjacking` `MCP` `Sentry` `编码代理`
- 来源：Tenet Security
- 链接：https://tenetsecurity.ai/blog/agentjacking-coding-agents-with-fake-sentry-errors

### 3. Anthropic 通过 SpaceX 算力协议提高 Claude Code 和 API 限额

- 分类：资本与算力
- 优先级：中优先级
- 摘要：Anthropic 官方宣布与 SpaceX 达成算力合作，使用 Colossus 1 数据中心全部计算容量，新增超过 300MW、约 220,000+ NVIDIA GPU 级别容量，并提高 Claude Code 五小时限额和 Claude Opus API 限额。
- 推荐理由：AI 产品体验正在被算力供给直接塑造，Claude Code 的使用限额变化也会影响开发者对 Agent 工具的日常依赖。
- 标签：`Anthropic` `Claude Code` `SpaceX` `算力`
- 来源：Anthropic
- 链接：https://www.anthropic.com/news/higher-limits-spacex

## 趋势观察

“AI 的下一阶段，不只是更强模型，而是更清晰的权限、治理和执行边界。”
