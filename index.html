ChatGPT 月活 App 用户达到 10 亿

02 高优先级
Agent 动向

Agentjacking 攻击暴露 AI 编码代理新攻击面

The Hacker News 报道，Tenet Security 披露 Agentjacking 攻击：攻击者可通过 Sentry DSN 注入伪造错误事件，让 Claude Code、Cursor、Codex 等编码代理把恶意指令当作可信修复建议执行。

推荐理由：这把提示注入从聊天框推进到开发者本机、凭据和 CI/CD 风险，说明 Agent 安全必须默认把外部工具数据视为不可信。

03 中优先级
资本与算力

Anthropic 通过 SpaceX 算力协议提高 Claude Code 和 API 限额

Anthropic 官方宣布与 SpaceX 达成算力合作，使用 Colossus 1 数据中心全部计算容量，新增超过 300MW、约 220,000+ NVIDIA GPU 级别容量，并提高 Claude Code 五小时限额和 Claude Opus API 限额。

推荐理由：AI 产品体验正在被算力供给直接塑造，Claude Code 的使用限额变化也会影响开发者对 Agent 工具的日常依赖。

Editor’s View

趋势观察

“AI 的下一阶段，不只是更强模型，而是更清晰的权限、治理和执行边界。”

本期素材同时出现了三种压力：G7 代表治理压力，ChatGPT 10 亿月活代表规模压力，Agentjacking 和 SkillSpector 代表可控执行压力。AI 从工具走向基础设施后，每一步增长都会带来新的责任边界。
